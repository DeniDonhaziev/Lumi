# UniFind — Поиск университетов для абитуриентов

Сайт для абитуриентов с поиском и фильтрацией университетов России.

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Возможности

- Поиск по названию вуза, городу и специальности
- Фильтры: город, тип вуза, специальность, проходной балл, общежитие
- Сортировка по рейтингу, баллам и названию
- Карточки с информацией о бюджетных местах, проходных баллах и направлениях

## Сборка

```bash
npm run build
npm run preview
```
